# 46009361 OS

A webOS hosted on Neocities, made for the Neocities Credits prize in the Flavortown shop. Whenever I see a sidequest, I always check what I can do to ensure the prize or work I'd achieve pays off. For example, when I shipped [Get Flavortown Ship ID](https://flavortown.hackclub.com/projects/12093), I didn't *need* to make my Chrome extension a Flavortown one, but I remembered seeing the instructions to register on the Explore [Extensions menu](https://flavortown.hackclub.com/explore/extensions) and decided that made the most sense.

Features:
* Window resizing
* Mobile dragging
* Ubuntu file explorer, in "My Computer"
   * The embedded resource goes to a domain that still works with Windows Vista Internet Explorer over plain HTTP
* `www.youtube-nocookie.com` embed
* Respects default browser theme
* A fun fact on a "safety net" feature I'd add if this were a real OS!
* Tipping, click "hosted by neocities"
